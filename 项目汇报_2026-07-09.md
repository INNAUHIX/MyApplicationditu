# 鸿蒙足迹记录 App — 项目汇报（2026-07-09）

> 包名 `com.footprintapp.pro`，显示名「足迹」（`$string:app_name`），HarmonyOS 6.1.1，设备仅 phone。

## 一、在跑的东西（entry 模块，唯一启动入口）
- `EntryAbility` 加载 `pages/Index`（main_pages 仅注册此页）。
- `Index.ets`（约 1439 行）单文件承载全部功能：
  - 地图 MapKit（3D 建筑、昼夜模式）
  - 实时定位 `geoLocationManager` + WGS84→GCJ02 转换
  - RDB 持久化 `footprint.db` / `track_points(id,lat,lng)`，落库 + 重启恢复
  - 足迹算法：位移 ≥80m 记一个"足迹"点，全部点连轨迹；展示两种：折线轨迹 polyline / 海量点 MassPointOverlay（手搓红色圆点 PixelMap）
  - 权限流程：前台(模糊+精确) → 后台定位引导
  - 底部 Tab（HdsTabs）：足迹 / 我的；"我的"半模态内手写 设置页 + 隐私设置页（多数为 "功能不可用" 占位）
- `EntryAbility` 额外做后台保活（`onBackground` 起 location 后台任务）
