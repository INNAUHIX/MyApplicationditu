# 第三阶段：红色轨迹记录功能实施计划

## 一、需求

完全按照用户要求：
1. **深色地图** - 使用官方 `dayNightMode` 设置
2. **自动记录轨迹** - 有定位就自动记录，不需要开始/停止按钮
3. **红色轨迹线** - 使用官方 `MapPolyline` 组件
4. **清除轨迹按钮** - 点击一键清除

---

## 二、使用的官方组件

### 2.1 深色模式（官方属性）
```typescript
mapOptions = {
  dayNightMode: mapCommon.DayNightMode.DARK
}
```

### 2.2 地图折线（MapPolyline - 官方组件）
**官方文档**：https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/map-polyline

```typescript
// 添加折线
mapController.addPolyline(polylineOption)

// 更新折线路径
mapPolyline.setPoints(points)

// 删除折线
mapPolyline.remove()
```

### 2.3 坐标转换（官方方法）
```typescript
map.convertCoordinateSync(WGS84, GCJ02, location)
```

### 2.4 连续定位（官方API）
```typescript
geoLocationManager.on('locationChange', requestInfo, callback)
```

## 三、实现方案

### 3.1 文件结构
只修改一个文件：`entry/src/main/ets/pages/Index.ets`

### 3.2 核心改动

**新增状态**：
- `trackPolyline: map.MapPolyline | null` - 轨迹线对象（官方组件）
- `trackPoints: mapCommon.LatLng[]` - 轨迹点数组

**新增方法**：
- `onLocationUpdate(location)` - 位置更新时自动添加轨迹点
- `clearTrack()` - 清除轨迹（调用官方 remove() 方法）

**新增UI**：
- 底部一个"清除轨迹"按钮

### 3.3 自动记录逻辑
1. 应用启动 → 定位权限申请
2. 授权成功 → 开启连续定位（Location Kit）
3. 位置更新 → 坐标转换 → 添加到轨迹点 → 更新折线（官方 setPoints）
4. 点击"清除轨迹" → 删除折线（官方 remove）→ 清空轨迹点

## 四、具体实现

### 步骤1：修改 Index.ets

1. **添加状态变量**：
```typescript
private trackPolyline: map.MapPolyline | null = null;
private trackPoints: mapCommon.LatLng[] = [];
```

2. **添加深色模式**：
```typescript
this.mapOptions = {
  dayNightMode: mapCommon.DayNightMode.DARK,
  // ... 其他配置
};
```

3. **修改 enableMyLocation**：开启连续定位并自动记录轨迹
```typescript
private async enableMyLocation() {
  // ... 现有代码 ...
  
  // 开启连续定位（官方API）
  geoLocationManager.on('locationChange', {
    priority: geoLocationManager.LocationRequestPriority.ACCURACY,
    scenario: geoLocationManager.LocationRequestScenario.NAVIGATION,
    timeInterval: 1,
    distanceInterval: 0
  }, async (location) => {
    // 坐标转换（官方方法）
    let latLng = await map.convertCoordinateSync(
      mapCommon.CoordinateType.WGS84,
      mapCommon.CoordinateType.GCJ02,
      location
    );
    
    this.trackPoints.push(latLng);
    
    // 如果还没有轨迹线，创建红色折线（官方组件）
    if (!this.trackPolyline) {
      this.trackPolyline = await this.mapController.addPolyline({
        points: this.trackPoints,
        width: 10,
        color: 0xFFFF0000,  // 红色
        geodesic: false
      });
    } else {
      // 更新轨迹线（官方方法）
      this.trackPolyline.setPoints(this.trackPoints);
    }
  });
}
```

4. **添加清除轨迹方法**：
```typescript
private async clearTrack() {
  if (this.trackPolyline) {
    await this.trackPolyline.remove();  // 调用官方方法
    this.trackPolyline = null;
  }
  this.trackPoints = [];
}
```

5. **添加底部按钮**：
```typescript
Button('清除轨迹')
  .backgroundColor('#333333')
  .width(120)
  .height(40)
  .borderRadius(20)
  .onClick(() => { this.clearTrack(); })
```
